Model X

Conozca el Model X

El Model X es el vehículo deportivo todoterreno más seguro, rápido y capaz de la historia. Gracias a su tracción en todas las ruedas y una batería de 100 kWh que proporciona una autonomía de 475 kilómetros, el Model X posee un amplio espacio para siete adultos sentados y todo su equipaje. Además, es locamente rápido, ya que acelera de cero a 100 kilómetros por hora en tan solo 3,1 segundos. El Model X es el SUV sin concesiones.

Asientos y almacenamiento

El Model X tiene el mayor espacio de almacenamiento de cualquier vehículo utilitario deportivo de su clase, y capacidad hasta para siete adultos. Está diseñado para maximizar la comodidad del pasajero. En la opción de siete plazas, la segunda y tercera filas de asientos se pueden plegar hasta quedar rasas, creando así espacio de carga adicional. El Model X está disponible en tres configuraciones personalizadas de asientos, puede ser optimizado para satisfacer tus necesidades específicas y las de tu familia.

Hardware para conducción autónoma en tu Model X

Todos los vehículos Tesla producidos en nuestra fábrica, incluido el Model 3, tienen el hardware necesario para la capacidad de conducción autónoma con un nivel de seguridad sustancialmente mayor que el de un conductor humano.
